🚀 COMMENT UTILISER?
════════════════════════════════════════════════════════════════════════════

OPTION 1: Script Console (2 min, pas d'installation)
─────────────────────────────────────────────────────
  1. Va sur: https://www.vinted.fr/member/items/favourite_list
  2. Ouvre: 1-vinted-console-script.js
  3. Copie tout le contenu
  4. Ouvre la console: Ctrl+Shift+J (Chrome) ou Cmd+Option+J (Mac)
  5. Colle le code et appuie Entrée
  6. Attends ~15 secondes
  7. Un panneau rouge apparaît → utilise-le!

OPTION 2: Extension Chrome (5 min, installation une fois)
──────────────────────────────────────────────────────────
  1. Va sur: chrome://extensions/
  2. Active le "Mode de développeur" (en haut à droite)
  3. Clique "Charger l'extension non empaquetée"
  4. Sélectionne le dossier: extension/
  5. C'est prêt! Chaque visite à Vinted, le bouton apparaît
